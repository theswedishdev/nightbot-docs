# Hug
Give a hug to a user.

## Usage
`!hug [<username>]`

#### Example usage
`!hug oatmealemily`

```
/me all the hugs for you oatmealemily <3 I appreciate you!
```

## User level
The recommended user level for this command is `everyone`.

## Add command
```js
!addcom !hug /me all the hugs for you $(touser) <3 I appreciate you!
```
